All the sketches are grouped in index.html

# Setup

If you don't have Node.js installed, used [fnm](https://github.com/Schniz/fnm) to install it.


Once installed, install packages:
```
npm install
```

# Dev

Start Vite server, open up URL in browser

```
npm run start
```

# Building

Generate static files ready for deployment to Github pages et al.

```
npm run build
```

To test static build, open up 'docs' folder in VS Code's _Live Server_.
